# My Kart

A simple, responsive e-commerce frontend prototype built entirely with vanilla HTML, CSS, and JavaScript. 

## Features
* Mock login page with password toggle.
* Dynamic product rendering, search, and category filtering.
* Working shopping cart with quantity adjustments and total calculation.
* Mock checkout flow and order history tracking.
* Responsive design for both desktop and mobile.

## How to Run
Since this is a purely frontend project, no local server or backend setup is required.

1. Clone this repository: `git clone https://github.com/yourusername/my-kart.git`
2. Open the folder and double-click `index.html` to run it in your web browser.

## Tech Stack
* HTML5
* CSS3
* JavaScript (Vanilla)
